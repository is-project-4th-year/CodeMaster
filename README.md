## Getting Started

Follow these steps to set up the project locally:

1. **Fork the repository** (optional but recommended):
   [CodeMaster GitHub Repository](https://github.com/is-project-4th-year/CodeMaster)  

2. **Clone your fork**:
   ```bash
   git clone https://github.com/<your-username>/CodeMaster.git
   cd CodeMaster
   ```

3. **Set up environment variables**:
   ```bash
   cp .env.example .env.local
   ```
   Then, open `.env.local` and add your **Supabase keys**.

4. **Install dependencies**:
   ```bash
   npm install
   ```

5. **Run the development server**:
   ```bash
   npm run dev
   ```

6. **Have fun learning!**
